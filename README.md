# 🧠 SmartLife — Personal Insights API

SmartLife is a REST API built with **FastAPI** that transforms personal data into **actionable insights**.  

It allows users to log personal metrics such as **sleep, productivity, steps, mood, or any custom metric**, and automatically generate analytics including **trends, correlations, anomaly detection, and visualizations**.

The goal of this project was to build a **scalable analytics backend for time-series personal data**, demonstrating how data pipelines and statistical analysis can generate meaningful insights from everyday information.

---

# 🚀 Technologies

This project was built using the following technologies:

- 🐍 **Python**
- ⚡ **FastAPI**
- 🗄️ **SQLAlchemy**
- 💾 **SQLite**
- 🔢 **NumPy**
- 📊 **Matplotlib**
- 🧩 **Pydantic**
- 🖥️ **Uvicorn**

---

# ✨ Features

## 📈 Track Personal Metrics

You can log time-series data such as:

- 😴 Sleep hours  
- 🏃 Exercise duration  
- 🧠 Productivity levels  
- 😊 Mood tracking  
- 👟 Daily steps  
- 🔥 Calories burned  
- 📊 Any custom metric  

Each metric is stored as a **timestamped data point**.

---

## 📊 Data Analytics

SmartLife automatically computes insights such as:

- 📉 Summary statistics  
- 📈 Trend analysis  
- 🔄 Rolling averages  
- 🔗 Correlation analysis between metrics  
- 🚨 Statistical anomaly detection  

These insights help identify **patterns and relationships in personal data**.

---

## 📉 Data Visualization

SmartLife can generate charts directly from the API including:

- 📈 Time-series plots  
- 📊 Trend visualizations  
- 🔍 Correlation scatter plots  

Charts are dynamically generated using **Matplotlib**.
